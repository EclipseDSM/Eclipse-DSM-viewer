Eclipse DSM-Viewer

Project description:

DSM-Viewer Eclipse plugin is used to display Java code dependencies in behavior of Dependency Matrices, or DSM`s (see DS-matrices).
It will also provide several kinds of Eclipse integration such as code navigation, help pages, etc.

The first ready version of plugin uses the Dtangler library to count dependencies.

We expect that the next version will use Eclipse AST for that purposes and provide additional options:

  • cyclic dependencies detection
  • code navigation
  • matrices clusterization
